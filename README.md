# Medical Chatbot (MC)

Generated by the Solace Agent Mesh SDLC pipeline. App type: **Conversational AI Chatbot **.

## Run locally
```bash
pip install -r requirements.txt
uvicorn main:app --reload --port 8095
# open http://127.0.0.1:8095
```

## Or with Docker
```bash
docker build -t mc .
docker run -p 8095:8095 mc
```
